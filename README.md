## How to run this App?

1. clone or download this repository
2. install dependencies

```shell
	npm install
# or
	npm i
```

3. start

```shell
	npm run dev
```

## Note that you need to set up your personal [Mongo DB](https://Mongodb.com) or wait for the demo version online, which I'll bring ASAP.
